// Featured Project

CAPITAL SMART CITY
IOT DATA PIPELINE

A city-wide data infrastructure designed and built solo — unifying 3000+ heterogeneous devices across 7 zones into a single AWS-hosted pipeline that powers the city's digital twin.

Go MQTT LoRaWAN Modbus AWS Kafka PostgreSQL Digital Twin
CompanySmart TechOne · FDH
RoleSolo Architect
SinceApril 2024
Status● Active · Scaling
Sites5 Residential · 2 Industrial
Scale3000+ Devices · 10K+ msg/day

The Challenge

Vendor-siloed dashboards, no unified view, no path to a digital twin. Handed a problem with no existing architecture and no engineering team.

The Build

Solo-designed Go ingestion service normalizing multi-protocol, multi-vendor payloads into clean JSON on AWS — serving structured data to the full downstream stack.

The Outcome

All vendor dashboards replaced. Digital twin enabled. New vendors and zones actively onboarding. Alerting, analytics and 3D city model in active development.
